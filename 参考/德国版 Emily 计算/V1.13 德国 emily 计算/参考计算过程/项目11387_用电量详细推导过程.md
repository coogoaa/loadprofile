## 2. 年总量

```text
base    = BASE["BY"] = 3302 kWh
t_base  = HVAC["heat_pump"] = 2000 kWh
t_ext   = t_base × usage.am
        = 2000 × 1.6
        = 3200.0 kWh

ev_ext  = ev_km × EV_KWH_PER_KM
        = 20000 × 0.18
        = 3600.0 kWh

final   = base + t_ext + ev_ext
        = 3302 + 3200.0 + 3600.0
        = 10102.0 kWh/年

davg    = final / 365
        = 27.6767 kWh/日
```

结论：项目 11387 当前年用电量为 **10,102.0 kWh/年**，日均 **27.68 kWh/日**。

## 3. 月度拆分

月度分布分三部分相加：

```text
sm[m]  = 1 + (制冷月 ? cmm - 1 : 0) + (制热月 ? hmm - 1 : 0)
rb[m]  = base × DE_MONTHLY[m] × sm[m]
srb    = Σ rb[m] = 4355.07384
nb[m]  = rb[m] / srb × base

tw[m]  = (制冷月 ? DE_MONTHLY[m] × cmm : 0)
       + (制热月 ? DE_MONTHLY[m] × hmm : 0)
stw    = Σ tw[m] = 1.14972
ta[m]  = tw[m] / stw × t_ext

ea[m]  = ev_ext / 12 = 300.0
fkm[m] = nb[m] + ta[m] + ea[m]
```

## 4. 小时拆分

小时分布先得到非 EV 的日内形状，再叠加 EV 充电形状：

```text
DAY_HOURS      = H09-H17
COOL_PEAK      = H14-H19
HEAT_PEAK      = H06-H09, H18-H22

om[h]  = h 在 DAY_HOURS ? occ_v : 1.0
pm[h]  = (h 在 COOL_PEAK ? cpm : 1) × (h 在 HEAT_PEAK ? hpm : 1)
adj[h] = DE_HOURLY[h] × om[h] × pm[h]
sadj   = Σ adj[h] = 1.127094
ns[h]  = adj[h] / sadj

dne    = (base + t_ext) / 365 = 17.8137 kWh/日
dev    = ev_ext / 365 = 9.8630 kWh/日
nek[h] = dne × ns[h]
evk[h] = dev × EV_PROFILE["mostly_overnight"][h]
fkd[h] = nek[h] + evk[h]
```
