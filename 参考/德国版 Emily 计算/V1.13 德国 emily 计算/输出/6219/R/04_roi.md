## 3. 年现金流公式

```
baseline_cost[t] = load_total × buy × (1+inf)^(t-1) + fixed_daily × 365
remain_cost[t]   = import × buy × (1+inf)^(t-1) + fixed_daily × 365
export_income[t] = export × sell                  # 馈网价不通胀
saving[t]        = baseline_cost - remain_cost + export_income
cf[0]            = -sysCost
```
