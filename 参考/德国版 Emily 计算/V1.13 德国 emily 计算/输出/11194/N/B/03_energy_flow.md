## 4. 电池模拟（日循环 · 月初 SOC=0）

```
usable_capacity = bat × DoD = 13.50 × 0.9 = 12.15 kWh
for each h ∈ 0..23:
    direct    = min(gen[h], load[h])                  # 直接消纳
    surplus   = gen[h] - direct                       # 剩余可充电
    deficit   = load[h] - direct                      # 缺口待补
    charge    = min(surplus, usable_capacity - soc)   # 充电
    soc      += charge
    discharge = min(deficit, soc × RTE)               # 放电
    soc      -= discharge / RTE
    export[h] = surplus - charge                      # 馈网
    import[h] = deficit - discharge                   # 购电
```
