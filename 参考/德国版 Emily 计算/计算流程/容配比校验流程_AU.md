以下是 AU 版本的 容配比校验流程参考

- 计算流程
1. 初选：
- 方案目标逆变器功率 = PV 总容量 / 目标容配比
- 目标容配比默认参数为 1.8
2. 匹配：
在相位允许的规格库，寻找>=目标功率的最小规格
3. 校验与调整：
计算 实际容配比 = PV总功率 / 选定逆变器功率

Case 1 (合规): 容配比 ≤ 200%。 -> 通过。

Case 2 (过高 > 200%):
- 优先尝试增大逆变器规格。
- 若已达相位允许的最大规格 (单相不超过 10kW)，则触发削减面板逻辑。
- 削减逻辑：计算允许的最大 PV 功率 (逆变器功率 * 2.0)，从最低分坡面开始减少面板，直到满足要求。

3.1.1.7 电池容量推荐
1. 策略说明：
- 基于预设的各州用电量、每小时用电比例，推导电池容量需求
- A 方案
E_req = max(整夜需求, aSurplus × 光伏剩余)
Battery_calc = E_req / 效率
Battery_nominal = 标准化(Battery_calc)

其中：
- 整夜需求：17:00-07:00（14小时）
- aSurplus：0.7
- 效率：预设的 Dod × RTE = 0.9 × 0.95 = 0.855
- B 方案
E_req = max(晚高峰需求, bSurplus × 光伏剩余)
Battery_calc = E_req / 效率
Battery_nominal = 标准化(Battery_calc)

其中：
- 晚高峰需求：17:00-21:00（4小时）
- bSurplus：0.55
- 效率：0.855
- C 方案
E_req = 晚高峰需求
Battery_calc = E_req / 效率
Battery_nominal = 标准化(Battery_calc)

其中：
- 晚高峰需求：17:00-21:00（4小时）
- 效率：0.855

3.1.1.9 完整推导过程示例：
示例 1：大屋顶（27.72kW）
步骤 1：屋顶坡面信息
slope 4: 方位角 269.9°, 20 块面板
slope 5: 方位角 180.0°, 4 块面板
slope 6: 方位角 90.0°, 4 块面板
slope 7: 方位角 180.0°, 17 块面板
slope 12: 方位角 90.0°, 3 块面板
slope 13: 方位角 0.0°, 15 块面板
合计：63 块面板
步骤 2：物理限制分析
有效坡面数量：6 个
物理最大容量：63 片面板
对应总功率：63 × 0.44 = 27.72 kW

步骤 3：用户用电需求计算（NSW 州）

年用电量：7,778 kWh
日均用电：7,778 / 365 = 21.31 kWh
晚高峰占比：22.1%
晚高峰用电：21.31 × 22.1% = 4.72 kWh
整夜用电：12.69 kWh

步骤 4：方案 A 推导

目标：物理极限满铺

初始配置：63 片 × 440W = 27.72 kW

逆变器初选：
目标功率 = 27.72 / 1.8 = 15.4 kW
选择：10 kW（单相最大）

容配比校验：
实际容配比 = 27.72 / 10 = 277%（超过 200%）
需要调整！

削减面板：
最大允许 PV = 10 × 2.0 = 20.00 kW
需要削减：27.72 - 20.00 = 7.72 kW → 18 片面板
从发电量最小的坡面顺次削减

最终配置：
PV：45 片 × 440W = 19.80 kW
逆变器：10 kW
容配比：198%（符合要求）

发电计算：
年发电量 = 19.80 × 1,460 = 28,908 kWh
日均发电 = 28,908 / 365 = 79.20 kWh
日均用电：7,778 / 365 = 21.31 kWh
白天用电：日均用电 × 43.88% = 9.35 kWh
光伏剩余 = 79.2 kWh - 9.35 kWh = 69.85 kWh

电池容量计算：
整夜需求 = 12.69 kWh
0.7 × 剩余 = 0.7 × 69.85 = 48.90 kWh
E_req = max(12.69, 48.90) = 48.90 kWh
Battery_calc = 48.90 / 0.855 = 57.19 kWh
Battery_nominal = 50 kWh（达到上限）

步骤 5：方案 B 推导

目标：13.0 kW（屋顶容量 > 15kW）

面板削减：
需要面板 = CEILING(13.0 / 0.44) = 30 块
从发电量最小的坡面开始削减

最终配置：
PV：30 片 × 440W = 13.20 kW
逆变器：8 kW（目标 7.3kW，选择 8kW）
容配比：165%（符合要求）

发电计算：
日均发电 = 13.20 × 1,460 / 365 = 52.80 kWh
光伏剩余 = 42.39 kWh

电池容量计算：
晚高峰需求 = 4.72 kWh
0.55 × 剩余 = 0.55 × 42.39 = 23.31 kWh
E_req = max(4.72, 23.31) = 23.31 kWh
Battery_calc = 23.31 / 0.855 = 27.26 kWh
Battery_nominal = 30 kWh

步骤 6：方案 C 推导

目标：6.6 kW

面板削减：
需要面板 = CEILING(6.6 / 0.44) = 15 块

最终配置：
PV：15 片 × 440W = 6.60 kW
