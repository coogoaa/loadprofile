# 批量项目数据下载与分析工具使用说明

## 功能概述

`batch_download_and_analyze.py` 脚本可以批量处理多个项目，完成以下任务：

1. ✅ 下载项目的 GIS 数据（detect_building.json）
2. ✅ 区分 with_pv 状态（True/False）
3. ✅ 下载 image_drawed 图片
4. ✅ 下载 buildingbox 图片
5. ✅ 提取面积信息（area）
6. ✅ 提取 is_old 状态
7. ✅ 关联 CSV 表格数据
8. ✅ 生成 Excel 汇总表格

## 使用方法

### 基本用法

```bash
cd "V1.7 测试/脚本"
python3 batch_download_and_analyze.py [项目ID列表] [选项]
```

### 参数说明

- **项目ID列表**（必需）：空格分隔的项目ID，例如：`6201 6219 6527`
- `--output-dir`：输出目录，默认为 `../测试数据`
- `--csv-file`：CSV 文件路径，默认为 `../CSV/V1.9版本 - 测试环境（官）(临时复制）_副本.csv`
- `--excel-output`：Excel 输出路径，默认为 `输出目录/batch_summary_时间戳.xlsx`

## 使用示例

### 示例 1：处理单个项目

```bash
cd "V1.7 测试/脚本"
python3 batch_download_and_analyze.py 6201 --output-dir ../测试数据
```

### 示例 2：处理多个项目

```bash
cd "V1.7 测试/脚本"
python3 batch_download_and_analyze.py 6201 6219 6527 6525 --output-dir ../测试数据
```

### 示例 3：指定 CSV 文件和输出路径

```bash
cd "V1.7 测试/脚本"
python3 batch_download_and_analyze.py 6201 6219 \
  --output-dir ../测试数据 \
  --csv-file ../CSV/V1.9版本\ -\ 测试环境（官）\(临时复制）_副本.csv \
  --excel-output ../汇总报告/项目分析_20260401.xlsx
```

### 示例 4：从文件读取项目ID列表

如果项目ID很多，可以先创建一个文本文件：

```bash
# 创建项目ID列表文件
cat > project_ids.txt << EOF
6201
6219
6527
6525
6528
EOF

# 批量处理
cd "V1.7 测试/脚本"
python3 batch_download_and_analyze.py $(cat project_ids.txt) --output-dir ../测试数据
```

## 注意事项

1. **网络连接**：确保能访问 `file.greensketch.ai` 服务器
2. **依赖库**：需要安装以下 Python 库：
   ```bash
   pip3 install requests pandas openpyxl
   ```
3. **文件权限**：确保输出目录有写入权限
4. **CSV 文件**：如果 CSV 文件不存在，脚本仍会运行，但不会关联 CSV 数据
5. **下载失败**：如果某个文件下载失败，脚本会继续处理其他文件，失败的项会在表格中显示为空

## 故障排查

### 问题 1：SSL 连接错误

如果遇到 SSL 错误，参考之前的解决方案：

```bash
pip3 install "urllib3<2.0"
```

### 问题 2：找不到 CSV 文件

检查 CSV 文件路径是否正确：

```bash
ls -la "V1.7 测试/CSV/V1.9版本 - 测试环境（官）(临时复制）_副本.csv"
```

### 问题 3：Excel 文件无法打开

确保安装了 openpyxl：

```bash
pip3 install openpyxl
```

## 高级用法

### 只下载特定类型的项目

可以先运行脚本，然后根据 Excel 表格中的 with_pv 列筛选：

```bash
# 下载所有项目
python3 batch_download_and_analyze.py 6201 6219 6527 --output-dir ../测试数据

# 在 Excel 中按 with_pv 列筛选
# True: 储能扩容项目
# False: 新建系统项目
```

### 批量验证项目

下载完成后，可以批量运行验证脚本：

```bash
cd "V1.7 测试/脚本"

# 批量验证
for project_id in 6201 6219 6527; do
  python3 verify_system_composition.py \
    --project-id $project_id \
    --data-dir ../测试数据 \
    --output-dir ../验证报告
done
```
