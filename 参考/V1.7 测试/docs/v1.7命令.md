
## 方式 A：先进入脚本目录再运行（最推荐）


```
cd "V1.7 测试/脚本"
python3 download_data.py 5809 --output-dir ../测试数据
python3 verify_system_composition.py --project-id 5809 --data-dir ../测试数据 --output-dir ../验证报告

```
说明：

download_data.py 会把数据下载到：V1.7 测试/测试数据/5809/
verify_system_composition.py 会输出报告到：V1.7 测试/验证报告/系统组成验证_5809_xxx/

## 方式 B：不 cd，直接用脚本相对路径运行

```
python3 "V1.7 测试/脚本/download_data.py" 5811 --output-dir "V1.7 测试/测试数据"
```
