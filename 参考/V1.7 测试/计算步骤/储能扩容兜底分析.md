# 储能扩容兜底触发分析

## 1. 兜底触发条件

储能扩容时，面板固定不可削减，容配比超标只能增大逆变器。
当最大逆变器仍无法满足容配比 ≤ 200% 时，触发兜底。

```
兜底条件: 扩容容量 / 最大逆变器功率 > 200%
即: FLOOR(N × k) × 0.44 / max_inverter > 2.0

其中:
  N = 物理最大面板数
  k = 屋顶容量使用系数（当前值 0.6）
```

### 各方案最大允许扩容面板数

| 方案 | 最大逆变器 | 最大允许容量 | 最大允许面板数 |
|------|:---------:|:----------:|:------------:|
| 方案A | 10 kW | 20.0 kW | **45 块** |
| 方案B | 8 kW | 16.0 kW | **36 块** |
| 方案C | 8 kW | 16.0 kW | **36 块** |

> 方案A最大允许 45 块扩容面板（20kW），方案B/C最大允许 36 块（16kW）

## 5. 关键结论

### 兜底触发的临界物理面板数（按系数 k）

| 系数 k | B/C方案兜底临界 | A方案兜底临界 | 安全范围（全部通过） |
|:------:|:-------------:|:----------:|:------------------:|
| 0.20 | N ≥ 185 | N ≥ 230 | **N ≤ 184** |
| 0.25 | N ≥ 148 | N ≥ 184 | **N ≤ 147** |
| 0.30 | N ≥ 124 | N ≥ 154 | **N ≤ 123** |
| 0.35 | N ≥ 106 | N ≥ 132 | **N ≤ 105** |
| 0.40 | N ≥ 93 | N ≥ 115 | **N ≤ 92** |
| 0.45 | N ≥ 83 | N ≥ 103 | **N ≤ 82** |
| 0.50 | N ≥ 74 | N ≥ 92 | **N ≤ 73** |
| 0.55 | N ≥ 68 | N ≥ 84 | **N ≤ 67** |
| 0.60 ← 当前 | N ≥ 62 | N ≥ 77 | **N ≤ 61** |
| 0.65 | N ≥ 57 | N ≥ 71 | **N ≤ 56** |
| 0.70 | N ≥ 53 | N ≥ 66 | **N ≤ 52** |

### 核心发现

1. **当前系数 k=0.60**：物理面板 ≤60 块时三套方案全部安全，≥61 块时 B/C 开始兜底，≥77 块时 A 也兜底
2. **系数越大，兜底越早触发**：k=0.70 时仅需 53 块就触发 B/C 兜底
3. **系数越小，安全范围越大**：k=0.20 时需要 181 块才触发 B/C 兜底（实际不太可能）
4. **方案A比B/C更耐高容量**：因为方案A有 10kW 逆变器可选，而 B/C 最大只有 8kW
5. **k=0.45 是一个较好的平衡点**：安全范围扩大到 ≤81 块，覆盖绝大多数住宅屋顶

# 方案B与C电池差异化分析

## 6. B=C 的根本原因

```
方案B: E_req = max(晚高峰需求, bSurplus × 光伏剩余)
方案C: E_req = 晚高峰需求

当 bSurplus × 光伏剩余 ≤ 晚高峰需求 时:
  → B 的 max() 取了晚高峰需求，与 C 完全相同
  → 电池一样大 → 逆变器一样（共用面板）→ 投资金额也一样
```

**B≠C 的充要条件**: `bSurplus × 光伏剩余 > 晚高峰需求`

即: `光伏剩余 > 晚高峰需求 / bSurplus`

> 注意：即使数值上 B > C，如果两者向上取整到同一个电池规格（5/6.5/10/13/15/19.5/26/30/39/45/50 kWh），B 和 C 仍然相同。

---

## 7. 差异化临界公式

```
光伏剩余 ≈ 扩容容量(kW) × 每kW日均发电(kWh) × (1 - 日间自用率)

B≠C 条件:
  bSurplus × FLOOR(N×k) × 0.44 × 每kW日发电 × (1-日间自用率) > 晚高峰需求

简化为:
  FLOOR(N×k) > 晚高峰需求 / (bSurplus × 0.44 × 每kW日发电 × (1-日间自用率))
```

## 12. 综合结论

### B=C 出现的典型场景

| 条件 | 说明 |
|------|------|
| 小屋顶 + 储能扩容 | N ≤ 20, k=0.60, bSurplus=0.45 → 几乎必然 B=C |
| 高晚高峰需求 | 晚高峰 > 8kWh 时，需要更多面板才能差异化 |
| 低日照州 | VIC 比 QLD 更难差异化 |

### 让 B≠C 的调整方向

| 调整方式 | 效果 | 副作用 |
|---------|------|--------|
| **增大 bSurplus** (0.45→0.55) | 直接降低差异化门槛 | B 电池偏大，可能过度推荐 |
| **增大 k** (0.60→0.70) | 更多面板→更多光伏剩余 | 兜底更早触发（53块 vs 61块） |
| **两者同时调整** | 效果叠加 | 需要平衡兜底风险 |

### 推荐平衡点

对于储能扩容场景，如果希望在大多数情况下 B≠C：
- **bSurplus=0.50, k=0.55** — 差异化门槛适中，兜底安全范围 N≤67
- **bSurplus=0.55, k=0.50** — 差异化更容易，兜底安全范围 N≤73
- 当前 **bSurplus=0.45, k=0.60** — 小屋顶(≤20块)基本都是 B=C
