## 能量流模拟与自用率计算

### 计算系统的总发电量
- 逻辑： 准确计算出整套太阳能系统在一年内可以发多少电，并将其细分到每个月的每一小时。
- 动作与公式：
  1. 正常流程:
    - 将每个区域的发电量数据加和
    - 公式 (年总发电量):
总年发电量 = Σ [ (单个屋顶坡面的单块面板年发电量) × (该坡面安装的太阳能板数量) ]
    - 公式 (每小时发电量):
总月度小时发电矩阵[月][小时] = Σ (各坡面的月度小时发电矩阵[月][小时])
  2. 兜底流程:
    - 触发条件： 只要有一个屋顶坡面的发电量计算失败 就会启用此方案。
    - 公式 (年总发电量):
估算年总发电量 = 系统总功率(kW)  × 系统配置的年均发电系数(kWh/kW) 
    - 细分到小时 
      - 使用兜底数据的月度发电比例和小时发电比例。
      - 公式 (某月某小时发电量):
发电量(月, 小时) = 估算年总发电量 × 月度发电比例(月) × 小时发电比例(小时)


### 能量流模拟与自用率计算
- 结合返回的光照辐射数据和小时发电量以及用电量情况，确定有多少电是自用的，多少需要买，多少可以卖。
- 以“天”为单位进行计算:
  1. 循环遍历一天24小时:
    - 发电量(小时): 从第2步的结果中获取。
    - 用电量(小时): 有各州月日剧用电量比例表格和各州各时段用电比例
  2. 计算关键小时数据:
    - 公式: 直接自用电量(小时) 
直接自用电量(小时) = min(发电量(小时), 用电量(小时))
    - 公式: 可用于充电的电量(小时) 
可用于充电的电量(小时) = max(发电量(小时) - 用电量(小时), 0)
  3. 汇总成“天”的数据:
    - 日总发电量 = Σ 发电量(小时)
    - 日总用电量 = Σ 用电量(小时)
    - 日总直接自用电量 = Σ 直接自用电量(小时)
    - 日总可充电量 = Σ 可用于充电的电量(小时)
    - 日非发电时段用电量 = 日总用电量 - 日总直接自用电量
  4. 计算最终有效充电量:
    - 逻辑: 电池充的电必须是晚上能用得上的才算有效。所以，一天中最终给电池充的电量，不能超过电池容量，也不能超过晚上（或无光时段）的总用电需求。
    - 公式: 最终有效充电量(日)
最终有效充电量(日) = min(日总可充电量, 电池可用容量, 日非发电时段用电量)
  5. 汇总年度数据:
    - 将每日数据乘以当月天数得到月度数据，再累加12个月得到年度数据。
    - 年总自用电量 = Σ (日均直接自用电量 + 日均最终有效充电量) × 当月天数 
    - 年总发电量 = Σ (月发电量)
    - 年卖电量 = 年总发电量 - 年总自用电量
  6. 计算自用率 
    - 公式:
每个月的自用率 = 月自用电量（月自发自用 + 月有效充电）/ 月发电量
自用率 = 年总自用电量 / 年总发电量
